Motivos

Dentro da computação existe diversas aplicações que podemos trabalhar uma delas é a programação voltada para sistemas web. Nessa disciplina o aluno vai ter experiência em trabalhar com várias ferramentas, como o gitHub ideal para controle de versão, e vários tipos de tecnologias e arquiteturas para ajudar no seu desenvolvimento. Ao fim da disciplina espera-se que o aluno esteja mais preparado para buscar se aprofundar em outras novas tecnologias que nascem todos os dias para esse mercado.

Informações adicionais

  • Disciplina de 4 créditos
  • Pré-requisitos: Gerência da Informação
  • O aluno é avaliado por três meios: laboratórios práticos, minitestes e um projeto
  • Também é chamado de Design de Software
